<h3>ITCell Help Page</h3>

<h4>Input Fields</h4>

<ul>
<li><a name="mhc"></a><b>MHC type:</b> it is possible to pick the type from the drop down menu or upload your own peptide-MHC structure (or model) in PDB
format. </li>

<li><a name="tcrfile"></a><b>TCR structure:</b> Please upload TCR structure (or model) in PDB format.
</li>

<li><a name="sequence"></a><b>Antigen sequence:</b> paste the sequence in FASTA format.</li>


<li><a name="email"></a><b>e-mail address:</b> the link with the
results of your request is sent to this address.</li>

<li><a name="jobname"></a><b>Job name: </b> the server generates a default name based on the time stamp. If user has provided a job name, the server will use it as a prefix for the job name.</li>
</ul>
